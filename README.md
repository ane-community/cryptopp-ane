cryptopp-ane
============

CryptoPP Adobe AIR Native Extension

[![Build Status](https://travis-ci.org/ane-community/cryptopp-ane.png?branch=master)](https://travis-ci.org/ane-community/cryptopp-ane)

## Prerequisites

* Adobe AIR SDK
* MingW

## Building

* cd library/cryptopp && build
* cd library && build
* gradle
